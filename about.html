CS498 - Trends in Surface Temperature

Exploring the how surface temperatures have changed over the last 50 years

Objective

The objectvie of this visualization is to provide the user insight on the changing surface temperatures in the last 50 years.

Structure

The structure of this project is an interactive slide show

The slides is driven by the author with navigation buttons indicating the order to proceed with

Then each page after the overview allows for the user to interact with the data to see different representation of it

Scene Utilizations

Each slide of the scene utilizes a consitant format and colloring scheme. Additionally the navigation highligted with arrows are similiar on each page to ensure the user can easily get confortale with navigation through the deck.

On each slide the ability to hover over items to obtain more insight is consitant as is the format and content of the data selection drop down bar on the 2nd and 3rd slide
